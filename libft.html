Introduction to headers files

Header files in C are files containing declarations of functions, variables, and types that are shared across multiple source code files in a program. They have a ".h" extension and are included at the beginning of source code files using the #include preprocessor directive.

Introduction to Makefile

A Makefile is a script used in software development to automate the building and compiling of a program from its source code. It contains a set of rules and instructions that specify how source files are compiled and linked together to create executable binaries.
