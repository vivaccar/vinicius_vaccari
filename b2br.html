BORN 2 BE ROOT

This project consists of creating a virtual machine and implementing an operating system following specific rules.

What is a Virtual Machine?

A virtual machine is a software-based emulation of a physical computer that operates like an independent system. It allows users to run multiple operating systems and applications on a single physical machine.

LVM

It is a system that allows for the dynamic allocation and management of storage on Linux systems. With LVM, you can create logical volumes that span multiple physical disks, resize volumes on-the-fly, and perform snapshots for backups. It offers flexibility and ease of management for storage resources, allowing administrators to better utilize disk space and simplify storage management tasks.

App Armor

AppArmor is a security feature in Linux that helps protect your system by controlling what resources and actions individual applications can access. It works by creating profiles for each application, which define what it can and cannot do. This helps prevent malicious actions and enhances system security.

Apt and Aptitude

APT (Advanced Package Tool) and Aptitude are both package management tools for Debian-based Linux distributions like Ubuntu. APT is more commonly used for scripting and automated package management tasks, while Aptitude is preferred for interactive package management by users who prefer a text-based interface.

SSH

SSH, or Secure Shell, is a cryptographic network protocol that enables secure communication and remote access between computers over an unsecured network. It provides a secure channel for executing commands, transferring files, and managing remote systems by encrypting all data exchanged between the client and server, thus preventing eavesdropping and unauthorized access.

UFW

UFW (Uncomplicated Firewall) is a user-friendly command-line tool for managing firewall rules on Linux systems, particularly those based on Ubuntu. It provides a simplified interface for configuring and managing firewall settings, allowing users to easily create rules to allow or block incoming and outgoing network traffic. With UFW, administrators can quickly and effectively secure their systems by defining rules to control which connections are permitted or denied.

SUDO

Sudo, short for "superuser do," is a command-line utility in Unix-like operating systems that allows users to execute commands with the privileges of another user, typically the superuser (root). It enables users to perform administrative tasks or run commands that require elevated permissions without needing to log in as the root user directly.

Monitoring Script

The task involves creating a bash script named "monitoring.sh" that displays specific system information on all terminals every 10 minutes upon server startup. The script must not show any errors and should include details such as the operating system architecture and kernel version, number of physical and virtual processors, available RAM and memory utilization rate, processor utilization rate, last reboot date and time, LVM activation status, active connections, number of users, server's IPv4 address and MAC address, and the number of commands executed with sudo. The script should utilize the wall command for displaying information and can optionally include a banner.
